User guide for Quiz App

The folder quiz/uceskai contains all required files to run the quiz mobile application. The main purpose of this app is the tracking of the app user location, and the subsequent popup of a question once they are in the near vicinity of that question's position. Furthermore, this app should return let the user pick an answer out of 4 options, submit said answer, save this in a database and return information on whether the selected answer was correct.
FEATURES
A question pop-up field: Read the question and select one of the 4 answer options underneath. Press "Submit" to send off your reply and receive feedback. Please note that by submitting your answer choice you agree to have your reply and user ID stored in the quiz database.

An interactive leaflet map: Press "Track my location" to navigate the map interface to your position. Other blue markers show the locations of quiz questions available. Zoom in or out of the map by clicking on + and -. To move the map content right click anywhere within the map and drag into the desired direction.

INSTALLATION
To install and run the application, clone the repository by entering in the terminal window:
git clone https://github.com/mkainz/quiz.git

Find the required server file httpServer.js in the repository github.com/mkainz/server.git and save it to your machine.
Navigate to the location of the server and start the server by typing:
node httpServer.js &
ctrl + c

Then navigate into the folder "uceskai":
cd uceskai

Start the phonegap app:
phonegap serve

Open a browser and type in your host followed by your port number, e.g.:
http://developer.cege.ucl.ac.uk:31299/

Once you are finished using the application, close the browser tab, navigate to your command lind and press
ctrl + c
